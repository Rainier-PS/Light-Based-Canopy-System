Light-Based Canopy System

An interactive miniature canopy system using Arduino with light-responsive technology

Project Overview

The Light-Based Canopy project is a miniature adaptive canopy system that responds to ambient light using Arduino. Constructed from recycled cardboard, servo motors, LDR, and LED lighting, this prototype demonstrates smart architecture concepts and is ideal for STEM exhibitions, interactive showcases, and educational demonstrations of light-responsive systems.

Materials & Tools

  • Arduino Uno microcontroller
  • Light-dependent resistor (LDR sensor)
  • LCD display module
  • LED lighting components
  • Servo motor for canopy movement
  • Breadboard & jumper wires
  • Cardboard sheets
  • USB cable or power bank
  • Basic DIY tools (glue, scissors, markers)

Assembly Process

  1. Design the canopy structure using cardboard for the frame
  2. Mount the Servo motor, Light Dependent Resistor (LDR), and LED indicators
  3. Connect all electronic components and sensors to the Arduino using a breadboard
  4. Write and upload the Arduino sketch for ambient light detection and response
  5. Test the system by varying light intensity and observing the canopy's response
